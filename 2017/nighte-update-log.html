nighte 更新日志（第一次用 nighte）

2017-08-20

既然是第一次用 nighte，当然是要截个图做纪念咯，截什么好呢？让我想想。

这是刚写自动【自动传图 & 自动 Github 提交】的脚本时遇到的问题。

即 cd /abc/x y z 这种带空格的目录怎么办，那上图就给出了方法，把变量用双引号引起来就好了。

其实 nighte 有几个彩蛋。

  • 一是有一套黑色主题的，我的想法是根据时间自动切换，如果用户手动点击了，就放到 cookie 里面保存起来，不再自动切换。
  • 以及 nighte 还实现了一个功能，就是每篇 markdown 文章的文件里面都可以加入自动移的 script 和 style，觉得如果要写一些比较具备专题性的文章，比如介绍 switch 上的某款游戏啦，把 background-image 换成游戏的主题背景不也挺好玩的吗。

最后，截一张我写这篇文章的书写环境，为此我还故意添加了代码功能 ：）

我想这样各位会看得更一目了然。
