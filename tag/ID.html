注册 ID

2008年 01月 03日

話說現在在網絡中申請ID是件很麻煩的事情，不是未滿6位數不然註冊，就是想了一個十分理想的名字被註冊了。這是相當懊惱的事情。要是剛出來的服務商比如百度有啊，飯否之類的還好，碰到Yahoo，Gmail就想都別想了，不說AlexChen這樣的沒有了，就是AlexChen1960這樣偏門的都被註冊了。 Orz 當然如果不是完美主義的話，走個偏偏偏門，那麼AlexChen199438560一般還是讓你給註冊的。

但是這種名字:

一來，除了自己能記住外基本上別人看一遍是記不住的。不利於網路上的交流，就好比玩一遊戲，大家聊過天，可是忘了交換名片，第二次有事情想找，記不住名字，就算知道前面叫AlexChen後面的數字誰記得住呀。

二來，在就是名字太長會影響註冊一些服務提供商的帳號，比如有的網絡提供商之允許用戶名在5-12個字符之間註冊。少了多了都不能註冊。

所以這次ID的註冊的確讓我花了不少心思去想，起初叫GouKen，GouKen啊Capcom公司街霸Ryu和Ken的師傅，死牛死牛的角色，在街霸4裡面還是隱藏Boss。可是每次我搜搜gouken的時候。 Google都提示我:*您要找的是不是狗啃。 *。 。 Orz 給google這樣一折騰，我崩潰了。咱還是換吧，不然到了以後用著用著不爽又得換。然後就想啊想。每次想我都會開Gmail的註冊界面去驗證要註冊的ID是否被註冊。據我經驗，一般能註冊Gmail的ID基本上沒什麼問題。基本上世界上不會再有第二個人用了。那麼接下來的Yahoo，MSN。 Twitter，就都可以註冊了。其實註冊MSN我是為了@Live。 com這個郵箱。用這個來登錄MSN，現在的MSN改為Windows Live Messenger成為Windows Live的一部分了。可是由於歷史遺留問題，我們還是叫它MSN，想想看如果按官方的叫法得叫WLM多難聽啊。 。然後註冊Yahoo郵箱是為了Flickr，Flickr是世界上最好的相冊。免費用戶可以上傳200張圖片。 200張意味著什麼?如果用心去維護這個相冊，只把拍到的照片精挑細選的upload上去。還是很夠用的。而且上面的​​氣氛很好，很多國外攝影好手。能收到他們對自己照片的評價是一件非常愉快的事情。

可是放棄使用Gouken這個ID後。到了第二天，還是沒想好新的。就要崩潰了。最後立馬睡覺，不想了，想睡醒一覺好好想。每次Gmail驗證的時候都說我註冊了。而且當我輸入多次的時候會跳出驗證碼讓我輸入。想想看那驗證碼。 。做得歪歪扭扭的，用來防止機器註冊的機制跑到我頭上。我眼睛都崩了。到了第三天忽然間想到MGS(METAL GEAR SOLID)的主角，SolidSnake，對!就是那個被稱為[傳說中的傭兵]的角色，然後我找到先前比較喜歡的一個名字Zoro，Zorro的不規範寫法，其實也規範啦，讀音是一樣的。可是zooro不助於記憶，就中國而言，zoro無疑是會念就會寫的，而Solid也基本上很多人都知道意思會寫，固體的意思。

按我的理解SolidZoro這個名字的意思是: 真佐羅。 。 Oh my god，請寬恕我的無知- -其實zoro不應該翻譯成佐羅，這裡只是音譯而已，就像真三國無雙 講求究極奧義。 。好吧，我再一次上演了自己的無知。 x_x。

哇噢，一個ID NAME寫了那麼多。我是一個廢話比較多的孩子，請各位見諒。

OK.SolidZoro is my Finally English Name.so I must cherish.
